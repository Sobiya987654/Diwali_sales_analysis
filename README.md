## 📌 Project Overview
This project focuses on **Exploratory Data Analysis (EDA)** of Diwali sales data to understand customer purchasing behavior and uncover meaningful business insights.  

The goal is to help businesses:
- Improve customer experience  
- Identify target audience  
- Increase sales through data-driven decisions  

---

## 🎯 Objectives
- Perform data cleaning and preprocessing  
- Analyze customer demographics  
- Identify purchasing patterns  
- Visualize key trends and insights  

---

## 📂 Dataset
The dataset contains information about:
- Customer details (Gender, Age, Marital Status, State)
- Product categories
- Purchase amounts  

---

## 🛠️ Tools & Technologies
- **Python**
- **Pandas** – Data manipulation  
- **NumPy** – Numerical operations  
- **Matplotlib & Seaborn** – Data visualization  

---

## 🔍 Key Analysis Performed
- ✔️ Data cleaning (handling null values, removing duplicates)  
- ✔️ Gender-wise sales analysis  
- ✔️ Age group purchasing behavior  
- ✔️ State-wise sales distribution  
- ✔️ Marital status impact on buying patterns  
- ✔️ Product category performance  

---

## 📊 Key Insights
- Female customers contribute more to overall sales  
- Age group **26–35 years** shows highest purchasing power  
- Certain states dominate sales performance  
- Married customers tend to spend more  
- Specific product categories drive maximum revenue
